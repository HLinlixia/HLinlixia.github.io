Liunx命令

发表于 2024-07-12

Linux 提供了一个强大的命令行接口，可以用于执行各种任务，如文件管理、进程控制、系统监控等。以下是一些常用的 Linux 命令及其用途：

文件管理

  1. 列出目录内容

    • -l：长格式列出目录内容
    • -a：显示所有文件，包括隐藏文件
    • -r，–reverse：逆序排列目录内容

  2. 改变工作目录

  3. 显示当前目录路径

  4. 创建目录

    • -p，–parents：如果目录已存在，不报错，且按需创建父目录

  5. 删除文件

    • -R，-r，–recursive：递归地删除目录及他们内容
    • -f，–force：强制删除，忽略不存在的文件和参数，不提示

  6. 复制文件或目录

    • -R，-r，–recursive：递归地复制文件
    • -f，–force：如果有一个已存在且无法打开的目标文件，将其删除并重试

  7. 移动或重命名文件或目录

    • -f，–force：覆盖前不提示

  8. 改变文件时间戳

    将文件的访问时间和修改时间更新为当前时间，除非提供 -c 或 -h 参数，否则程序会在文件不存在时创建一个空文件

文本操作

  1. 连接文件并在标准输出上输出

  2. 类似于 cat ，用于分页阅览文件

  3. more 的增强版，支持翻页和搜索

  4. 查看文件的开头部分

    将文件列表中的每个文件的起始10行内容输出至标准输出

  5. 查看文件的结尾部分

    在标准输出上显示每个文件的最后10行。

    • -f，–follow：当文件增长时，输出后续添加的数据

检索和查找

  1. 在指定目录下查找文件和目录

    • -name pattern：按照文件名查找，支持使用通配符*和？
    • -type type：按照文件类型查找，可以是f（文件），d（目录），l（符号链接）等
    • -size [+-]size：按照文件大小查找，+/-表示大于或者小于指定大小，单位可以是c（字节），w（字数），d（块数），k（KB），M（MB），G（GB）
    • -mtime days：按修改时间查找，+/-表示在指定天数前/后，days表示天数

  2. 查找文件中符合条件的内容

    在文件 [files] 范围内查找满足条件 pattern 的内容

    • -i：忽略大小写
    • -n：显示匹配内容的行号
    • -r：递归查找子目录中的文件

  3. 参数传递过滤器，命令组合工具

    给命令传递参数，常用来组合多个命令

  4. 查找文件

    在环境变量 $PATH 设置的目录里查找符合条件的文件

  5. 查找文件

    用于查找源码，二进制文件或者帮助文件

    • -B：查找二进制文件
    • -S：查找源码文件
    • -M：查找说明文件

用户和权限

  1. 创建用户账户

    用于创建用户，一般使用useradd创建好用户之后，再用passwd设定账户的密码

  2. 创建或修改密码

  3. 删除用户账户

vim/vi操作

正常模式

  • h：光标左移。
  • j：光标下移。
  • k：光标上移。
  • l：光标右移。
  • w：移动到下一个单词开头。
  • b：移动到前一个单词开头。
  • 0：移动到行首。
  • $：移动到行尾。
  • gg：移动到文件开头。
  • G：移动到文件结尾。
  • :n：移动到第n行。
  • u：撤销。
  • Ctrl + r：重做。

插入模式

  • i：在光标前插入。
  • I：在行首插入。
  • a：在光标后插入。
  • A：在行尾插入。
  • o：在当前行下插入新行。
  • O：在当前行上插入新行。
  • Esc：返回Normal模式。

可视模式

  • v：进入可视模式。
  • V：进入行可视模式。
  • Ctrl + v：进入块可视模式。
  • y：复制选中区域。
  • d：删除选中区域。
  • c：更改选中区域。

命令行模式

  • :w：保存文件。
  • :q：退出。
  • :q!：强制退出不保存。
  • :wq或:x：保存并退出。
  • :e filename：打开文件。
  • :w filename：另存为指定文件。
  • :set number：显示行号。
  • :set nonumber：隐藏行号。
  • :syntax on：开启语法高亮。
  • :syntax off：关闭语法高亮。

删除、复制、粘贴

  • x：删除光标处字符。
  • dd：删除整行。
  • ndd：删除游标所在的下面n行
  • yy：复制整行。
  • nyy：复制游标所在的下面n行
  • p：在光标后粘贴。
  • P：在光标前粘贴。

搜索与替换

  • /pattern：向前搜索模式。
  • ?pattern：向后搜索模式。
  • n：跳到下一个匹配。
  • N：跳到上一个匹配。
  • :%s/old/new/g：全文件替换。
  • :s/old/new/g：当前行替换。
